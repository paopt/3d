## 三角函数运算
sin(a+b) = sin(a)*cos(b) + sia(b)*cos(a)
sia(a-b) = sin(a)*cos(b) - sin(b)*cos(a)

cos(a+b) = cos(a)*cos(b) - sin(a)*sin(b)
cos(a-b) = cos(a)*cos(b) + sin(a)*sin(b)

sin(-a) = -sin(a)
cos(-a) = cos(a)


## 矢量运算
### 点乘，就是数量积，
angle为ab向量的夹角
a = (a1, a2)
b = (b1, b2)
a * b = |a| * |b| * cos(angle)
a * b = a1*b1 + a2*b2 + a3*b3

### 叉乘，就是矢量积, 三维
a * b 得到第三个矢量，即ab平面的法向量
a = (x1, y1, z1)
b = (x2, y2, z2)
a X b = (y1z2 - y2z1, x2z1-x1z2, x1y2-x2y1)
